SECTION 2 — SPATIAL ORIENTATION MODEL

Every location is modeled as a 360° cylinder space.

Define:
FRONT VIEW = 0°
BACK VIEW = 180°

Left and right seam boundaries must match spatially.

Define:
LEFT EDGE of Front == RIGHT EDGE of Back
RIGHT EDGE of Front == LEFT EDGE of Back

Both images must share:

Same horizon line height

Same camera height

Same perspective depth

Same lens focal logic

Same lighting direction

============================================================
SECTION 3 — CONSERVATION LAW (앞/뒷면 보존 법칙)

Unique Dominant Object Rule
If a dominant object appears in FRONT, it must NOT repeat in BACK.

Example:
If FRONT contains:

Bed (center dominant)

BACK must NOT contain:

Another bed

Mirrored bed

Duplicate furniture

Architectural Conservation
If FRONT shows:

Main palace entrance
Then BACK must logically show:

Inner garden depth

Mountain beyond

Exit gate

Secondary corridor
NOT another main entrance.

Access Logic Rule
Doors and windows must reflect plausible geometry.

If FRONT:

Door centered

Then BACK:

Must represent opposite interior wall

Door cannot reappear unless circular structure is defined

============================================================
SECTION 4 — CONNECTION SEAM CONSISTENCY RULE
(앞 뒷면 연결부분 일관성 해결)

At both side edges of image:

Define “Edge Anchors”

Edge Anchors must match across views:

Path continuation

Bush row

Fence alignment

Floor tile direction

Wall material

Column spacing

Lighting source direction

Ceiling beam rhythm

If FRONT left edge contains:

Stone curb + iron fence

Then BACK right edge MUST contain:

Same stone curb + iron fence continuation

No mismatch allowed:
Example forbidden:
FRONT edge: bush hedge
BACK edge: concrete curb

============================================================
SECTION 5 — OBJECT DISTRIBUTION LAW

Divide space into:
Foreground (0–3m)
Midground (3–15m)
Background (15m+)

If FRONT foreground contains major prop:
BACK foreground must be spatially coherent,
but must not duplicate that prop.

Dominant object types rotate:

Furniture

Architectural focal point

Natural focal point

Light source focal point

No mirroring symmetry unless location is symmetrical by design.

============================================================
SECTION 6 — DEPTH & HIERARCHY LOGIC

FRONT:
Primary narrative anchor.

BACK:
Context expansion anchor.

Example:
Palace Garden

FRONT:

Path leading toward palace entrance

Fountain midground

Palace in distance

BACK:

Same path continues

Garden gate or mountain horizon

Distant forest or town

Never:
Palace front in both directions.
